# 📘 GreatDocuments — плагин паспортной системы для Minecraft

GreatDocuments добавляет на ваш сервер полноценную систему документов: внутренние паспорта, заграничные паспорта и удостоверения о работе. Все документы представлены в виде красивых книг с цветным оформлением. Плагин поддерживает запросы на просмотр, автоматическое заполнение полей на основе удостоверения и удобное редактирование через команды.

## 🚀 Возможности

- **Внутренний паспорт** – личные данные, биография, прописка, информация о выдаче.
- **Заграничный паспорт** – упрощённый документ со ссылкой на внутренний паспорт.
- **Удостоверение о работе** – подтверждение трудоустройства в министерстве.
- **Взаимосвязь документов** – поля «Выдано в» и «Должность» в паспорте автоматически заполняются из удостоверения редактора.
- **Проверка полномочий** – редактировать чужие паспорта может только сотрудник, имеющий заполненное удостоверение.
- **Запрос на просмотр** – `/gd view <игрок> [passport|workpass]` с кнопками принятия/отклонения.
- **Автоматическое обновление дат** – при любом изменении документа обновляются дата выдачи и «Кем выдан».
- **Сообщения «Отсутствует»** – для незаполненных разделов книги (личные данные, биография, выдача, сроки).
- **Цветное оформление** – компоненты Adventure с цветами и рамками.
- **Таб-комплит** – подсказки для всех подкоманд и полей.

## 📦 Установка

1. Скачайте последний релиз плагина.
2. Поместите `.jar` в папку `plugins` сервера.
3. Перезапустите сервер (или используйте `/reload` с осторожностью).
4. При первом запуске автоматически создастся `config.yml`.

> **Требования:**  
> - Сервер Paper 1.16.5+ (или Spigot с поддержкой Adventure).  
> - Java 8 или выше.

## 🔧 Команды

Все команды начинаются с `/gd`.

| Подкоманда | Аргументы | Описание | Разрешение |
|------------|-----------|----------|-------------|
| `passport` | – | Открыть свой внутренний паспорт | — |
| `foreignpassport` | – | Открыть свой загранпаспорт | — |
| `workpass` | – | Открыть своё удостоверение о работе | — |
| `view` | `<игрок> [passport\|workpass]` | Запросить просмотр документа у игрока (по умолчанию паспорт) | — |
| `acceptview` | `<игрок>` | Принять запрос на просмотр | — |
| `denyview` | `<игрок>` | Отклонить запрос | — |
| `setpassport` | `[игрок] <поле> <значение>` | Изменить поле внутреннего паспорта (если игрок не указан – себе) | `greatdocuments.setpassport` |
| `setforeignpassport` | `[игрок] <поле> <значение>` | Изменить поле загранпаспорта | `greatdocuments.setforeignpassport` |
| `setworkpass` | `[игрок] <поле> <значение>` | Изменить поле удостоверения о работе | `greatdocuments.setworkpass` |

### Поля для `setpassport`

| Поле | Описание | Проверка |
|------|----------|----------|
| `гражданство` | Гражданство | текст |
| `возраст` | Возраст | целое число |
| `государство` | Государство (паспортная принадлежность) | текст |
| `номер` | Номер паспорта | текст |
| `пол` | Пол | `Мужской` / `Женский` |
| `рост` | Рост (см) | целое число |
| `месторождения` | Место рождения | текст |
| `семейноеположение` | Семейное положение | `Женат`, `Не женат`, `Замужем`, `Не замужем` |
| `супруг` | Имя супруга/супруги | текст |
| `страна` | Страна прописки | текст |
| `район` | Район прописки | текст |
| `улица` | Улица прописки | текст |
| `дом` | Номер дома | текст |
| `координаты` | Координаты места жительства | текст |

> **Примечание:** Поля «Выдано в» (министерство) и «Должность» не редактируются вручную – они автоматически берутся из удостоверения редактора.

### Поля для `setforeignpassport`

| Поле | Описание | Проверка |
|------|----------|----------|
| `гражданство` | Гражданство | текст |
| `возраст` | Возраст | целое число |
| `пол` | Пол | `Мужской` / `Женский` |
| `номер` | Номер загранпаспорта | текст |

### Поля для `setworkpass`

| Поле | Описание | Проверка |
|------|----------|----------|
| `местоработы` | Место работы (министерство) | текст |
| `должность` | Должность | текст |
| `возраст` | Возраст | целое число |
| `номерпаспорта` | Номер паспорта сотрудника | текст |
| `министерство` | Министерство, выдавшее удостоверение | текст |

## 🔐 Разрешения

| Разрешение | Описание |
|------------|----------|
| `greatdocuments.setpassport` | Доступ к `/gd setpassport` |
| `greatdocuments.setforeignpassport` | Доступ к `/gd setforeignpassport` |
| `greatdocuments.setworkpass` | Доступ к `/gd setworkpass` |

По умолчанию разрешений нет ни у кого. Операторы (OP) имеют доступ ко всем командам. Рекомендуется выдавать права через LuckPerms или аналогичный плагин.

## ⚙️ Конфигурация

Файл `config.yml` создаётся автоматически. Пример структуры:

```yaml
players:
  <UUID игрока>:
    passportState: "Гражданин"
    citizenship: "Россия"
    age: "25"
    gender: "Мужской"
    height: "180"
    birthplace: "Москва"
    maritalStatus: "Женат"
    spouse: "Анна"
    residence:
      country: "Россия"
      district: "Центральный"
      street: "Ленина"
      house: "15"
      coords: "x:100 y:64 z:200"
    passportNumber: "1234 567890"
    issueDate: "27.04.2026"
    validUntil: "27.05.2026"
    issuedBy: "Admin"
    ministry: "Министерство юстиции"
    issuerPosition: "Начальник отдела"
    foreignPassport:
      citizenship: "Россия"
      age: "25"
      gender: "Мужской"
      number: "AB123456"
      issueDate: "27.04.2026"
      validUntil: "11.05.2026"
      issuedBy: "Admin"
    workPass:
      workPlace: "Министерство юстиции"
      position: "Начальник отдела"
      age: "25"
      passportNumber: "1234 567890"
      ministry: "Министерство юстиции"
      issueDate: "27.04.2026"
      validUntil: "27.04.2027"
      issuedBy: "Admin"
```

> **Формат дат:** `dd.MM.yyyy`.  
> Сроки: внутренний паспорт – 1 месяц, загранпаспорт – 2 недели, удостоверение – 12 месяцев.

## 🧩 Механики и логика

### 🔗 Связь удостоверения и паспорта
- Поля `ministry` (в книге «Выдано в») и `issuerPosition` («Должность») в паспорте **нельзя изменить вручную**.
- При редактировании **чужого** паспорта они автоматически копируются из удостоверения редактора (`workPlace` и `position`).
- Если у редактора нет заполненного удостоверения, команда `/gd setpassport <игрок>` будет отклонена.
- При редактировании **своего** паспорта поля заполняются из собственного удостоверения (если оно есть), но их отсутствие не запрещает команду.

### 📝 Автоматическое обновление дат
- При любом изменении любого поля документа (паспорт, загранпаспорт, удостоверение) автоматически обновляются:
  - `issueDate` – текущая дата.
  - `validUntil` – дата окончания срока действия.
  - `issuedBy` – ник редактора (того, кто выполнил команду).

### 👀 Запрос на просмотр
- `/gd view <ник> passport` (или без указания типа – по умолчанию паспорт).
- `/gd view <ник> workpass` – показать удостоверение.
- Игрок, которому отправили запрос, получает сообщение с кнопками `[Посмотреть]` и `[Отказаться]`.
- Просмотр возможен только если оба игрока в одном мире и на расстоянии ≤ 15 блоков.
- Повторный запрос можно отправить не чаще, чем раз в 30 секунд.

### 📖 Оформление книг
- Все книги содержат декоративные рамки и цветовое оформление.
- Если раздел паспорта не заполнен (все значения равны «Не указано»), отображается крупная надпись «ОТСУТСТВУЕТ» / «ОТСУТСТВУЮТ».
- Загранпаспорт использует центрирование текста на титульной странице.

## 📖 Примеры использования

### 1. Получение удостоверения (для сотрудника)
```
/gd setworkpass местоработы Министерство юстиции
/gd setworkpass должность Начальник отдела
/gd setworkpass возраст 30
/gd setworkpass номерпаспорта 123456
/gd setworkpass министерство Министерство юстиции
```
Открыть удостоверение: `/gd workpass`

### 2. Заполнение своего паспорта
```
/gd setpassport гражданство Россия
/gd setpassport возраст 25
/gd setpassport пол Мужской
/gd setpassport государство Гражданин
/gd setpassport номер 1234 567890
/gd setpassport месторождения Москва
/gd setpassport семейноеположение Женат
/gd setpassport супруг Анна
/gd setpassport район Центральный
/gd setpassport улица Ленина
/gd setpassport дом 15
/gd setpassport координаты 100 64 200
```
Открыть паспорт: `/gd passport`

### 3. Редактирование чужого паспорта (требуется удостоверение)
```
/gd setpassport Steve семейноеположение Женат
/gd setpassport Steve супруг Alex
```
При этом в паспорте Стива автоматически пропишутся ваше место работы и должность.

### 4. Просмотр документов других игроков
- Запросить паспорт: `/gd view Steve`
- Запросить удостоверение: `/gd view Steve workpass`
- Принять запрос: нажать `[Посмотреть]` или ввести `/gd acceptview Steve`

## 🛠️ Для разработчиков

Плагин написан на Java с использованием Paper API и Adventure. Ключевые классы:

- `DocsBase` — главный класс плагина, регистрация команд.
- `PassportBook`, `ForeignPassportBook`, `WorkPassBook` — генерация книг.
- `PassportCommand` — обработка всех подкоманд, таб-комплит, логика запросов.
- `DocumentUtils` — утилиты (центрирование текста, обновление дат).

Данные хранятся в `config.yml` в секции `players.<UUID>`.
